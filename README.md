# Alarm Clock

## ⏰ Overview

Alarm Clock is a web-based alarm management application developed using HTML, CSS, and JavaScript. The application displays the current time in real-time and allows users to create alarms that trigger at specified times.

This project demonstrates fundamental JavaScript concepts including time and date manipulation, event handling, DOM updates, and audio notifications. It serves as an excellent learning resource for developers interested in building interactive time-based applications.

## ✨ Features

* Real-time digital clock display
* Set custom alarms
* Alarm notification system
* Audio alert functionality
* Multiple alarm support
* Responsive and user-friendly interface
* Lightweight and fast performance
* Browser-based operation without external dependencies

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser such as:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

### Installation

Clone the repository:

```bash
git clone https://github.com/frankpatel1/Alarm-Clock.git
```

Navigate to the project directory:

```bash
cd Alarm-Clock
```

Open the application:

```bash
index.html
```

Or simply open `index.html` in your preferred browser.

## 📖 Usage

1. Launch the application.
2. View the current time displayed on the clock.
3. Select the desired alarm time.
4. Set and save the alarm.
5. Wait for the specified time.
6. Receive an audio notification when the alarm is triggered.
7. Stop or reset the alarm as needed.
